Git

版本控制系统,SVN ,GIT

一个项目由多人开发

Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目.

可以有效、高速的处理从很小到非常大的项目版本管理。[2] Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

安装命令

yum -y install git
