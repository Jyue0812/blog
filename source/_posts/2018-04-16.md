#### 第0004题
第 0004 题： 任一个英文的纯文本文件，统计其中的单词出现的个数。

```
#打开文件读取内容，以空格分开形成列表
def open_file(path):
    with open(path, 'r') as fr:
        passage = fr.read()
    words = passage.split(' ')
    return words

#查找单词函数
def search(input_word, words):
    count = 0
    for word in words:
        if word == input_word:
            count += 1
    if count == 0:
        print('对不起，这篇文章没有这个单词')
    else:
        print('这个单词在本文共计出现%s次' %count)


#调用函数
path = 'book.txt'
if __name__ == '__main__':
    input_word = input('请输入一个英文单词')
    words = open_file(path)
    search(input_word, words)
```
